## 🧮 Problems Explained

### 1. **Trigonometry**
- **Question**: In a right triangle with sides 7, 24, 25 → Find `sec C + cot A`.  
- **Answer**: 4/3 ✅  

✔️ `template1_solved_1.html` → 7, 24, 25 → Answer = 4/3  
✔️ `template1_solved_2.html` → 8, 15, 17 → Answer = 5/3

### 2. **Compound Interest**
- **Question**: A sum of ₹12000 amounts to ₹20736 in 3 years at compound interest → Find the rate.  
- **Answer**: 20% ✅  

✔️ `template2_solved_1.html` → ₹12000 → ₹20736 → Rate = 20%  
✔️ `template2_solved_2.html` → ₹5000 → ₹6050 in 2 years → Rate = 10%
